# План монетизации rav2d

Дата: 2026-05-03

## План действий

```
1. Зарегистрировать имя "rav2d" на GitHub/crates.io       [сейчас, бесплатно]
2. PoC через c2rust                                        [2–4 недели]
3. Подать заявку в NLnet (дедлайн 1 июня 2026)             [€5K–50K]
4. Показать PoC в Prossimo/ISRG                            [$250K–$1M]
5. Параллельно: AOM, Sovereign Tech Fund                   [€50K–500K]
6. По мере зрелости: консалтинг для AOM-компаний           [$100–200/час]
```

## Уровни монетизации (по приоритету)

### Уровень 1: Гранты — самый проверенный путь

| Источник | Суммы | Подходимость |
|----------|------:|-------------|
| Prossimo / ISRG | $250K–$1M+ | Идеально — профинансировали rav1d |
| Sovereign Tech Fund | €50K–€500K+ | Отлично — финансируют FFmpeg |
| NLnet / NGI Zero | €5K–€50K+ | Хорошо — дедлайн 1 июня 2026 |
| AOM напрямую | Не раскрывается | Отлично — уже финансируют dav2d |
| FLOSS/fund | До $1M пул | Хорошо |

### Уровень 2: Консалтинг

- Интеграция AV2 для стриминговых компаний
- Оптимизация для SoC-вендоров
- Модель TwoOrioles / VideoLabs / FFlabs
- $100–200+/час

### Уровень 3: Наём в AOM-компанию

Компании которым нужен AV2 декодер:
- Google (Chrome, YouTube, Android)
- Meta (Instagram, WhatsApp)
- Netflix, Amazon Prime Video, Twitch
- Apple (Safari, FaceTime)
- Microsoft (Teams, Edge)

### Уровень 4: Security narrative

- Каждый CVE в C-декодерах → аргумент за Rust
- Chrome VRP: до $250K за RCE
- NSA рекомендует memory-safe языки

### Уровень 5: Hardware IP consulting

- Помощь SoC-вендорам с AV2 hardware decoders
- Софтварный декодер как reference implementation

## Окно возможности

- AV2 spec draft: февраль 2026
- Хардварных декодеров нет
- dav2d ещё ранний
- rav1d доказал модель
- Кто первый → тот получит финансирование

## Тайминг-риск

AV2 спецификация не финализирована. Пока spec меняется, код устаревает.
Митигация: следить за upstream dav2d, синхронизировать изменения.
