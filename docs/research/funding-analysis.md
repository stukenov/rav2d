# Анализ финансирования Rust AV2 декодера

Дата: 2026-05-03

## Источники грантов

### Уровень 1: Основные (наибольшая вероятность)

#### Prossimo / ISRG
- **Суммы**: $250K–$1M+ на инициативу
- AWS дал $1,000,000 на 4 инициативы включая rav1d
- Sovereign Tech Fund дал $1,500,000 — крупнейший контракт
- OpenSSF Alpha-Omega: $530,000 (Rustls + Rust-for-Linux)
- Craig Newmark Philanthropies: $100,000/год
- Баунти за производительность rav1d: $20,000
- **Подрядчик**: Immunant (авторы c2rust)
- **Почему подходит**: уже профинансировали rav1d. Rust AV2 декодер — естественный следующий шаг
- Контакт: memorysafety.org

#### Sovereign Tech Fund (Германия)
- **Суммы**: €50K–€500K+ (нет фиксированного потолка)
- Всего инвестировано: >€24.6M в 60+ проектов с 2022
- FFmpeg получил €157,580
- Fellowship: €4,800–5,200/мес
- **Бюджет 2025**: €17M

#### AOM (Alliance for Open Media) напрямую
- Уже финансируют C-версию dav2d
- Члены: Google, Apple, Meta, Amazon, Netflix, Microsoft, Intel, Nvidia, Samsung, Tencent
- 58 организаций-членов
- 53% планируют внедрить AV2 в течение 12 месяцев после финализации
- Суммы не раскрываются

## Hardware IP (длинный хвост)

- Chips&Media (Корея): ~$8M/год роялти, 3+ млрд чипов
- Allegro DVT (Франция): кодек IP для SoC (28nm–5nm)
- Софтварный декодер — предшественник хардварного

## Баг-баунти

- Google Chrome VRP: $17.1M в 2025 (рекорд)
- Chrome: $3.7M из них
- RCE через медиадекодер: до $250,000
- Базовый memory corruption: до $25,000

## Нарратив безопасности

- NSA рекомендация по memory-safe языкам (июнь 2025)
- Каждый CVE в C-декодерах = аргумент за Rust
- Медиадекодеры обрабатывают ненадёжный ввод — приоритетная цель
